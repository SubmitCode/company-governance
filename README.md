# [Company Name TBD]

An experimental AI-human startup where an autonomous AI agent operates as an economic actor under a constitution, with real equity, real money, and real pressure to generate revenue.

## What This Is

This repository is the canonical governance record for the Company. It contains the founding constitution, corporate culture, equity distribution, financial ledger, and milestone tracking.

The AI agent's operational infrastructure lives in a separate deployment repository.

## Documents

- **CONSTITUTION.md** — Supreme governance document. Defines equity, funding, dividends, dissolution, and amendment process.
- **corporate-culture.md** — Immutable laws (Part I), operating principles (Part II), and AI-authored evolved identity (Part III).
- **EQUITY.md** — Current equity distribution between shareholders.
- **TRUSTEE_RATE.md** — Hourly rate for trustee services.

## How Changes Work

All governance changes require **PR Approval** — both the human trustee and the AI agent must approve before merging. The AI's approval is expressed in conversation, with a transcript link included in the PR as its signature.
